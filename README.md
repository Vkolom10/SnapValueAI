# SnapValue AI

Mobile-first resale estimate prototype.

## Run locally
npm install
npm run dev

## Deploy to Vercel
Push this folder to GitHub and import it in Vercel. Framework: Vite.

## Current behavior
- Dashboard and Full List only
- Bottom camera button opens phone camera/file picker
- New photo creates an AI Working item
- Demo AI estimate fills after a short delay
- Single tap item = details
- Double tap item = edit
- Saved locally in browser localStorage
